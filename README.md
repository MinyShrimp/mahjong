## 프로젝트 설명
### 개요
엄청 옛날에 마작을 치고 군대때문에 잠깐 멈췄다가 최근들어 다시 치고 있는데, 문득 javascript를 이용하여 점수 계산기를 만들면 어떨까 생각하게 됐다. 

인터넷만 있으면 어디서든 접속할 수 있고(마장엔 와이파이가 설치되어 있다), 현재 다니고 있는 이수마장의 시스템에 딱 맞는 프로그램이 있다면 얼마나 좋을까 생각하다가 만들기 시작했다.
